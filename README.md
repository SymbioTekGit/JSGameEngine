# JAVASCRIPT GAME ENGINE (JSGE)

## Installation

Clone the git Repository
```
% git clone https://github.com/SymbioTekGit/JSGameEngine.git
% cd JSGameEngine
% npm install
```

Launching the Game.
```
% npm start
```

Connect to url: [http://127.0.0.1:8080/](http://127.0.0.1:8080/)
